Ce site à pour but de mettre en avant un restaurant fictif afin de pouvoir mettre en place un système de réservation
à partir d'un mail envoyé au restaurant afin de trasmettre la demande de réservation.

Pour ce faire ce site utilise la fonction "mail" de php mais qui présente un certains nombre de défauts pour cela une version avec la librairie PHP 
mailer à été imaginé mais n'est pas fonctionnel.

Arborescence:

Js contient tous les fichiers nécessaires pour la gestion du slide et du système de créneau (imposible de réserver avant l'heure qu'il est au moment de la consultation du site par l'utilisateur etc...)

Css permet de mettre en forme la totalité du site internet.

PHP s'occupe du traitement de la demande de réservation et de l'envoie du mail au restaurant.

Les fichiers HTML resprésentent les différentes pages du site internet.

The goal of this website is to showcase a fictional restaurant and implement a reservation system through an email sent to the restaurant to submit the reservation request.

To achieve this, the site uses PHP's "mail" function, which has certain limitations. Therefore, a version using the PHP Mailer library was considered, but it is not functional.

Structure:

Js contains all the necessary files for managing the slideshow and the time slot system (e.g., preventing reservations before the current time when the user views the site, etc.).

Css is used to style the entire website.

PHP handles the reservation request processing and sending the email to the restaurant.

HTML files represent the different pages of the website.
